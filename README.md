# icon-label

A polymer web component that shows an iron-icon and stylized text that is also optionally a link.

Example:
```html
        <icon-label
          icon="icons:polymer"
          icon-style-class="style-class"
          link="http://link"
          target="_blank"
          text="Click Me">
        </icon-label>
```

Custom mixins:
```html
        --icon-label-link-hover
        --icon-label-text
```

### Dependencies

Dependencies are installed using [Bower](http://bower.io/):

    npm install -g bower
    bower install
